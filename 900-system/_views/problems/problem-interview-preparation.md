## Планы по срокам

### 1 неделя (экстренная подготовка)
1. **День 1-2:** [[patterns-overview]] — 5 ключевых паттернов (sliding window, two pointers, BFS/DFS, binary search, DP basics)
2. **День 3-4:** [[system-design-android]] — мобильный system design
3. **День 5:** [[behavioral-questions]] — подготовить 5 историй по STAR
4. **День 6-7:** Mock interviews, ревью слабых мест

### 2 недели (нормальная подготовка)
1. **Неделя 1:** Coding — [[leetcode-roadmap]], по 3-4 задачи в день
2. **Неделя 2:** System design + behavioral параллельно
3. Ежедневно: 1 mock interview (даже с самим собой)

### 1 месяц (основательная подготовка)
1. **Неделя 1-2:** [[cs-fundamentals-overview]] + [[patterns-overview]] — полное покрытие
2. **Неделя 3:** [[architecture-distributed-systems]] + [[architecture-overview]]
3. **Неделя 4:** [[behavioral-interview]] + mock interviews
4. Параллельно: [[leetcode-roadmap]] — ежедневная практика
