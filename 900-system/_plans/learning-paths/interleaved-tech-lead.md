# Tech Lead / EM: интерливинговый путь обучения

> Для Senior-инженера, переходящего на роль Tech Lead или Engineering Manager.
> Четыре переплетённых трека: Лидерство, Коммуникация, Архитектура, Карьерный рост.

---

## Как пользоваться этим планом

- **Темп:** 2-3 файла в день, ~60-90 минут
- **Расписание:** ~5 дней в неделю, 8-10 недель
- **Интерливинг:** каждый день переключайтесь между треками (Leadership / Communication / Architecture / Career) — это повышает долгосрочное запоминание ([[desirable-difficulties]])
- **Review Day:** каждые 4-5 единиц — день закрепления. Перечитайте заметки, ответьте на вопросы, свяжите концепции
- **Чекбокс:** отмечайте `[x]` по мере прохождения

> [!tip] Принцип интерливинга
> Мозг лучше формирует связи, когда темы чередуются. Не группируйте — переключайтесь. Это сложнее, но эффективнее ([[how-brain-learns]], [[transfer-of-learning]]).

---

## Обзор недель

| Недели | Фокус | Треки |
|--------|-------|-------|
| 1-2 | Ясность роли + Основы коммуникации | Leadership (Roles) + Communication (Fundamentals) |
| 3-4 | Управление людьми + Навыки обратной связи | Leadership (Management) + Communication (Feedback, Difficult) |
| 5-6 | Построение команды + Архитектура + Презентации | Leadership (Team) + Architecture + Communication (Presentations, Written) |
| 7-8 | Найм + Стратегия + Переговоры | Leadership (Hiring, Strategy) + Communication (Negotiation) |
| 9-10 | Executive-навыки + Карьерный рост + Кросс-культура | Leadership (Executive, Org Design) + Career + Communication (Cross-Cultural) |

> [!tip] Почему архитектура сейчас?
