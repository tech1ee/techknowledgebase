# 300-content

> Контент для публикации: статьи, скрипты, материалы для соцсетей.

---

## Структура

```
300-content/
├── articles/           → Текстовый контент
│   ├── telegram/       → Посты для Telegram-канала
│   ├── blog/           → Статьи для блога/Habr
│   └── longread/       → Длинные аналитические материалы
│
└── scripts/            → Скрипты для медиа
    ├── video/          → Сценарии для YouTube/видео
    └── podcast/        → Скрипты для подкастов
```

---

## Форматы по типам

### Telegram (`articles/telegram/`)
- Короткие посты (до 4096 символов)
- Тизеры к видео/материалам
- Мысли и наблюдения
- Анонсы

### Blog (`articles/blog/`)
- Средний формат (1500-4000 слов)
- Туториалы и гайды
- Разборы технологий
- Карьерные советы

### Longread (`articles/longread/`)
- Глубокая аналитика (5000+ слов)
- Исследования рынка
- Масштабные обзоры

### Video Scripts (`scripts/video/`)
- Сценарии для YouTube
- Структура: hook → content → CTA
- Таймкоды и визуальные заметки

### Podcast Scripts (`scripts/podcast/`)
- Outline для эпизодов
- Ключевые тезисы
- Вопросы для гостей

## Принципы создания контента

**Из [[content-levels]]:**
- Минимум уровень 2 (кросс-тематические связи)
- Лучше уровень 3 (actionable инсайты)
- Идеально уровень 4 (открывает возможности)

**Из [[engaging-writing]]:**
- Фронтлоудинг: главное — сразу
- Короткие абзацы (2-4 строки)
- Открытые петли для удержания
- Паттерн-прерыватели каждые 300-500 слов
- Мосты между разделами

**Анти-паттерны:**
- Шаблонный AI-текст
- Избыточные цифры без контекста
- Банальные выводы
- "Мотивационная вода"
