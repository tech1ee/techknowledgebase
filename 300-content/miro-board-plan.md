# IT Market 2025: План Miro-доски

## Исследование возможностей

### Что можно через REST API:
- **Sticky Notes** — основной элемент для тезисов
- **Shapes** — блоки, стрелки, геометрия для диаграмм
- **Frames** — контейнеры для группировки секций
- **Connectors** — линии между элементами (требуют 2 endpoint'а)
- **Text** — текстовые блоки
- **Images** — можно загружать графики из visualization/output/
- **Tags** — для категоризации

### Что НЕ поддерживается API:
- Mind maps (создаются только вручную)
- Tables
- Code blocks
- Kanban boards

## Рекомендуемый план действий

### Шаг 1: Ручная структура в Miro
1. Создать новую доску
2. Использовать шаблон "Annual Report Presentation" или "Mind Map" как основу
3. Создать 6 основных Frame'ов (по разделам отчета)

### Шаг 2: Импорт графиков
Загрузить PNG из `visualization/output/`:
- 21-25: Увольнения
- 31-38: Зарплаты по категориям
- 39-40: Remote decline, RTO
- 41-42: Языки, рынки СНГ

### Шаг 3: Sticky Notes с ключевыми тезисами

**Frame 1: Кризис 2025**
- 165K уволенных
- +9% vs 2024
- Intel 25K, Microsoft 15K, Amazon 14K
- AI трансформация 20%
- Junior под ударом

**Frame 2: Remote умирает**
- 56% → 12%
- Amazon 5 дней
- 6x конкуренция
- Geographic arbitrage мертв

**Frame 3: Языки и рынки**
- Python AI/ML лучший выбор
- Go DevOps стабильный
- TypeScript Frontend избегать
- 1000+ заявок на позицию

**Frame 4: СНГ возможности**
- Kazakhstan 50K+ вакансий
- Sanctions: RU/BY blocked
- Outsourcing 30-40% комиссия
- Локальные рынки растут

**Frame 5: Пути по уровням**
- Junior → Local → Upskill
- Mid → Outsourcing
- Senior → Direct remote

**Frame 6: Action Items**
- Специализация > язык
- Timeline 18-24 месяца
- Реалистичные ожидания

### Шаг 4: Коннекторы
Соединить логические переходы между блоками стрелками

### Шаг 5: Цветовое кодирование
- Красный: Негативные тренды
- Желтый: Предупреждения
- Зеленый: Возможности
- Синий: Нейтральные факты

## Автоматизация через API (опционально)

Если хочешь автоматизировать создание элементов:

### Получение токена
1. Зайти на https://miro.com/app/settings/user-profile/apps
2. Создать новое приложение
3. Получить access token с scope `boards:write`

### Python скрипт для создания sticky notes

```python
import requests

MIRO_TOKEN = "your_token"
BOARD_ID = "your_board_id"

def create_sticky(content, x, y, color="light_yellow"):
    url = f"https://api.miro.com/v2/boards/{BOARD_ID}/sticky_notes"
    headers = {
        "Authorization": f"Bearer {MIRO_TOKEN}",
        "Content-Type": "application/json"
    }
    data = {
        "data": {"content": content, "shape": "square"},
        "style": {"fillColor": color},
        "position": {"x": x, "y": y}
    }
    response = requests.post(url, json=data, headers=headers)
    return response.json()

# Пример использования
stickies = [
    ("165K уволенных в 2025", 0, 0, "light_pink"),
    ("Remote: 56% → 12%", 300, 0, "light_pink"),
    ("AI/ML: +68% премия", 0, 300, "light_green"),
    ("Kazakhstan: 50K+ вакансий", 300, 300, "light_green"),
]

for content, x, y, color in stickies:
    create_sticky(content, x, y, color)
```

## Альтернативы Miro

Если нужно больше программируемости:
