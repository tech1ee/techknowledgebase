# Research Report: Intervals Pattern (2024-2025)

**Date:** 2025-12-29
**Sources:** 20+

## Executive Summary

Interval problems involve merging, inserting, and scheduling overlapping ranges. Core approach: **sort by start time, then merge**. Appears in 15-20% of coding interviews at FAANG.

## Key Insights

> "Sort intervals by start time, then iterate through them, merging overlaps as you go." — [Medium](https://medium.com/@piyushkashyap045/master-the-merge-intervals-pattern-your-secret-weapon-for-coding-interviews-aec54eb9023b)

> "If 'a' overlaps 'b' (b.start <= a.end), merge them into a new interval 'c'." — [Design Gurus](https://medium.com/codex/grokking-the-coding-interview-pattern-merge-interval-6e6b1e9e038c)
