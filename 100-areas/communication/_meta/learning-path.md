## Level 1: Foundation

**Продолжительность:** 2-3 недели
**Для кого:** Все уровни (Junior → Senior)

### Материалы

| # | Материал | Цель | Время |
|---|----------|------|-------|
| 1 | [[communication-models]] | Понять КАК работает коммуникация | 1 день |
| 2 | [[active-listening]] | Освоить базовый навык #1 | 3 дня |
| 3 | [[email-communication]] | Применять каждый день | 2 дня |

### Критерии готовности

□ Могу объяснить модель Shannon-Weaver
□ Применяю SOLER technique в 1-on-1
□ Использую BLUF в каждом email
□ Замечаю барьеры коммуникации в реальном времени

### Практика

```
НЕДЕЛЯ 1:
├── День 1-2: Прочитать communication-models.md
├── День 3-5: Прочитать active-listening.md
└── День 6-7: Практиковать в 1-on-1

НЕДЕЛЯ 2:
├── День 1-2: Прочитать email-communication.md
├── День 3-5: Применять BLUF во всех emails
└── День 6-7: Рефлексия, самооценка

НЕДЕЛЯ 3:
├── Закрепление всех навыков
└── Переход к Level 2
```

## Level 2: Core Skills

**Продолжительность:** 3-4 недели
**Для кого:** Middle+ (критично для code review)

### Материалы

| # | Материал | Цель | Время |
|---|----------|------|-------|
| 4 | [[giving-feedback]] | Давать конструктивную обратную связь | 3 дня |
| 5 | [[receiving-feedback]] | Принимать feedback без defensiveness | 2 дня |
| 6 | [[negotiation-fundamentals]] | Negotiate scope, deadlines, resources | 3 дня |

### Критерии готовности

□ Использую SBI framework в code review
□ Могу объяснить разницу между Radical Candor квадрантами
□ Знаю свой BATNA перед каждым negotiation
□ Не перехожу в defensive mode при получении feedback

### Практика

```
НЕДЕЛЯ 1:
├── День 1-3: Прочитать giving-feedback.md
├── День 4-5: Применить в 3+ code reviews
└── День 6-7: Получить feedback на свой feedback

НЕДЕЛЯ 2:
├── День 1-2: Прочитать receiving-feedback.md
├── День 3-5: Активно запрашивать feedback
└── День 6-7: Рефлексия

НЕДЕЛЯ 3-4:
├── Прочитать negotiation-fundamentals.md
├── Применить при обсуждении сроков/scope
└── Переход к Level 3
```

## Level 3: Advanced

**Продолжительность:** 4-6 недель
**Для кого:** Senior+ (критично для visibility)

### Материалы

| # | Материал | Цель | Время |
|---|----------|------|-------|
| 7 | [[conflict-resolution]] | Разрешать tech disagreements | 4 дня |
| 8 | [[technical-presentations]] | Доносить идеи до аудитории | 5 дней |
| 9 | [[stakeholder-negotiation]] | Работать с руководством | 3 дня |

### Критерии готовности

□ Знаю 5 стилей конфликтов (Thomas-Kilmann)
□ Провёл 1+ tech talk / demo
□ Успешно negotiate'd с stakeholder (scope/timeline)
□ Применял "disagree and commit"

### Практика

```
НЕДЕЛЯ 1-2:
├── Прочитать conflict-resolution.md
├── Идентифицировать текущие конфликты
└── Применить interest-based approach

НЕДЕЛЯ 3-4:
├── Прочитать technical-presentations.md
├── Подготовить и провести tech talk
└── Получить feedback

НЕДЕЛЯ 5-6:
├── Прочитать stakeholder-negotiation.md
├── Применить при следующем planning
└── Переход к Level 4
```

## Level 4: Leadership

**Продолжительность:** Ongoing
**Для кого:** Tech Lead / Manager

### Материалы

| # | Материал | Цель | Время |
|---|----------|------|-------|
| 10 | [[performance-conversations]] | Проводить performance review | 3 дня |
| 11 | [[delivering-bad-news]] | Сообщать плохие новости | 2 дня |
| 12 | [[saying-no]] | Отказывать без damage | 2 дня |
| 13 | [[cultural-dimensions]] | Понимать культурные различия | 2 дня |
| 14 | [[remote-team-communication]] | Управлять distributed team | 3 дня |

### Критерии готовности

□ Провёл 3+ performance review без эскалации
□ Сообщил плохую новость (задержка/отмена) без потери доверия
□ Сказал "нет" stakeholder с сохранением relationship
□ Адаптировал коммуникацию для разных культур

### Практика

Этот уровень — ongoing. Материалы изучаются по мере необходимости.
