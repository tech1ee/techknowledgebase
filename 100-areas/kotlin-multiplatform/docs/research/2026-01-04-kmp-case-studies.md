# Research Report: KMP Case Studies

**Date:** 2026-01-04
**Sources Evaluated:** 15+
**Research Depth:** Deep

## Executive Summary

Major companies с proven KMP success: Netflix (50% shared, studio apps), McDonald's (80%+ shared, 6.5M monthly purchases, fewer crashes), Cash App (7+ years production), Forbes (80%+ shared), Quizlet (migrated from JS, speed improvements), Philips (healthcare SDK). Ключевые метрики: 60-80% shared code типично, 40% faster development (Netflix), 60% bug reduction (McDonald's), 99%+ crash-free (Worldline).

## Key Findings

1. **Netflix**
   - ~50% code decoupled from platform
   - Studio apps for TV/movie production
   - Features shipped faster
   - 40% reduced feature development time

2. **McDonald's**
   - 80%+ shared application logic
   - 6.5M monthly purchases
   - 100M+ downloads
   - Fewer crashes, better performance
   - 60% reduction in platform-specific bugs

3. **Cash App**
   - 7+ years in production (since 2018)
   - Critical financial features
   - First FAANG company (Square/Block)
   - Developer happiness priority

4. **Forbes**
   - 80%+ shared logic
   - Simultaneous feature rollout iOS/Android

5. **Quizlet**
   - Migrated from JavaScript to Kotlin
   - Notable speed improvements
   - 100M active installs

6. **Philips**
   - Healthcare SDK (HealthSuite Digital Platform)
   - Faster feature implementation
   - Improved Android/iOS collaboration
   - Uses OpenAPI codegen

7. **Other Notable**
   - Bitkey (Block): 95% shared
   - Fast&Fit: 90%+ shared including UI
   - Worldline Eroski: 99%+ crash-free, 800K users
