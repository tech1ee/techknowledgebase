# Швейцария для разработчиков: €115K+ и качество жизни

Швейцария платит больше всех в Европе. Senior Software Engineer в Zurich: CHF 150,000-200,000 (€155,000-207,000). Это не опечатка. Но есть catch: стоимость жизни в Zurich — одна из самых высоких в мире. И виза сложнее чем в NL или DE: quotas, proof of "no EU candidate available".

## B Permit: Work Visa

### Типы permits

| Permit | Для кого | Срок |
|--------|----------|------|
| B Permit | Non-EU workers | 1-2 года, renewable |
| L Permit | Short-term (< 1 год) | 3-12 месяцев |
| C Permit | Permanent residence | После 5-10 лет B |

### Требования для B Permit (Non-EU)

```
1. Job offer от швейцарской компании

2. Labour market test
   → Компания должна доказать, что не нашла
     подходящего EU/EFTA кандидата
   → Для tech ролей обычно проходит

3. Specialized skills
   ✓ Software Developer = qualified
   ✓ Senior/Lead roles = easier approval

4. Quota system
   → ~8,500 B permits на всю страну в год
   → Распределяется по кантонам
   → Компании планируют заранее

5. Salary соответствует local standards
   → Senior dev: минимум CHF 110,000-130,000
```

### Процесс

```
Timeline: 4-12 недель

1. Получаешь job offer
2. Компания подаёт заявку в кантональный office
3. SECO review (labour market test)
4. Quota allocation check
5. Approval → Entry visa
6. Прибытие в Switzerland
7. Регистрация в муниципалитете
8. Получение B Permit card
```

### Сложности

```
⚠️ Quotas ограничены
   - Компании подают в начале года
   - К осени квоты могут закончиться

⚠️ Labour market test
   - Нужно доказать unique skills
   - Для senior roles обычно ок
   - Junior — сложнее

⚠️ Привязка к работодателю
   - Смена работы = новый permit
   - Но в рамках того же кантона проще
```

## Налоги

### Особенность: Canton matters

Налоги в Швейцарии федеральные + кантональные + муниципальные. Разница между кантонами огромная.

| Canton | Effective Tax Rate (CHF 150K income) |
|--------|-------------------------------------|
| Zug | ~15-18% |
| Schwyz | ~16-19% |
| Zurich | ~22-25% |
| Geneva | ~28-32% |
| Bern | ~24-28% |

### Пример Zurich

```
Gross: CHF 150,000

Federal tax: ~4,500
Cantonal tax: ~18,000
Municipal tax: ~8,000
Social security: ~9,000
───────────────────────
Total deductions: ~39,500

Net: ~CHF 110,500/year
Effective rate: ~26%
```

### Сравнение с EU

```
CHF 150,000 gross in Switzerland (Zurich):
Net: ~CHF 110,000 (€114,000)

€150,000 gross in Netherlands:
Net: ~€90,000 (с 30% ruling)
Net: ~€82,000 (без 30% ruling)

€150,000 gross in Germany:
Net: ~€85,000

Switzerland wins on net, even with higher costs.
```

## Практические шаги

### Если рассматриваешь Швейцарию

1. **Оцени timeline**
   - Quotas = нужно начинать рано (Q1 лучше)
   - Processing: 4-12 недель

2. **Target компании с experience**
   - Big Tech (Google, Meta, Apple)
   - Banking (UBS, Credit Suisse)
   - Они знают процесс, имеют quota allocation

3. **Negotiate правильно**
   - Minimum CHF 130K для senior
   - Relocation package важен
   - Sign-on bonus часто дают

4. **Документы**
   - Диплом (может потребоваться recognition)
   - CV в европейском формате
   - References

### Чеклист при переезде

```
□ Регистрация в муниципалитете (14 дней)
□ Health insurance (обязательна!)
□ Bank account (UBS, Credit Suisse, PostFinance)
□ Мобильный (Swisscom, Sunrise)
□ AHV number (social security)
```

## Плюсы и минусы

### Плюсы

```
✓ Максимальные зарплаты в Европе
✓ Высокое качество жизни
✓ Низкие налоги (vs EU)
✓ Политическая стабильность
✓ Центр Европы (легко путешествовать)
✓ Сильный tech sector
✓ Путь к permanent residence
```

### Минусы

```
✗ Очень дорогое жильё и жизнь
✗ Сложный visa process (quotas)
✗ Labour market test requirement
✗ Не EU (но есть bilateral agreements)
✗ Integration может быть сложной
✗ Landlords требуют высокий deposit (3 months)
```
