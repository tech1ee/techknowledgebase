# Нидерланды: быстрая виза, 30% tax-free, и Amsterdam tech scene

Kennismigrant (Highly Skilled Migrant) — одна из самых быстрых рабочих виз в Европе: 2-4 недели processing. Плюс 30% ruling: первые 5 лет 30% дохода не облагаются налогом. При зарплате €100K это экономит ~€9K/год. Amsterdam стал tech-хабом Европы: Booking, Adyen, Miro, Nike — все здесь.

---

## Терминология

| Термин | Что это |
|--------|---------|
| **Kennismigrant** | Highly Skilled Migrant visa |
| **30% ruling** | Tax benefit: 30% зарплаты tax-free |
| **IND** | Immigration and Naturalisation Service |
| **BSN** | Burger Service Number (аналог SSN) |

```

```

## Kennismigrant Visa

### Требования (2025)

```
1. Job offer от IND-recognised sponsor
   → Большинство tech компаний уже в списке
   → Проверь: https://ind.nl/en/public-register-recognised-sponsors

2. Salary thresholds (2025):
   ├── Age ≥ 30: €5,331/month gross
   ├── Age < 30: €3,909/month gross
   └── Recent graduates: €2,801/month gross

3. Роль = highly skilled
   ✓ Software Developer
   ✓ Android Engineer
   ✓ DevOps
   ✓ Data roles
```

### Процесс

```
Timeline: 2-4 недели (одна из самых быстрых в EU)

1. Подписываешь контракт с компанией
2. Компания (спонсор) подаёт заявку в IND
3. IND одобряет (2 недели обычно)
4. Получаешь MVV (entry visa) если нужна
5. Прибытие в NL
6. Регистрация в муниципалитете → BSN
7. Получение residence permit
```

### Документы

- Паспорт
- Трудовой контракт
- Диплом (legalized, иногда нужен перевод)
- Фото
- Antecedents certificate (справка о несудимости)

## 30% Ruling

### Eligibility (2025)

```
Требования:
1. Recruited from abroad (жил 150+ км от NL границы
   в 16 из 24 месяцев до найма)

2. Salary threshold:
   ├── Standard: €46,107/year
   └── Under 30 with Master's: €35,048/year

3. Specific expertise (для tech — автоматически)

4. Application within 4 months of start date!
   ⚠️ Критично: не пропусти дедлайн
```

### Что даёт

- **30% дохода tax-free** — как "extraterritorial costs"
- **Driver's license** — можно не менять на голландский
- **Box 3 exemption** — savings/investments не облагаются
- **5 лет** длительность (было 8, сократили)

### Подводные камни

```
⚠️ 4 месяца на подачу — пропустил = потерял навсегда

⚠️ При смене работы:
   - Новый работодатель должен подать заново
   - Перерыв > 3 месяцев = теряешь ruling

⚠️ Работа должна быть в NL
   - Remote на иностранную компанию не считается
```

### Сравнение с зарплатой

```
Senior Android: €85,000 gross
С 30% ruling net: ~€55,000/year = €4,580/month

Расходы (comfortable): €4,000
Savings: €580/month = €7,000/year

Без 30% ruling:
Net: ~€50,000/year = €4,167/month
Savings: €167/month = €2,000/year
```

### Housing crisis: главная проблема

```
Реальность:
- Waiting lists на social housing: 10+ лет
- Private rental: viewing с 20+ людьми
- Landlords требуют 3-4x rent в income proof
- Expats часто переплачивают первый год

Советы:
- Funda.nl, Pararius — основные сайты
- Facebook groups ("Expats in Amsterdam")
- Временно: furnished short-stay первые месяцы
- Рассмотри пригороды: Haarlem, Utrecht, Amstelveen
```

## Практические шаги

### Timeline переезда

```
Месяц 1: Offer + подписание
Месяц 1-2: IND processing (компания делает)
Месяц 2: MVV (если нужна) + прибытие
Месяц 2-3: BSN + bank account + housing search
Месяц 3-4: 30% ruling application (НЕ ПРОПУСТИ!)
Месяц 4-6: Settle into permanent housing
```

### Чеклист первых дней

```
□ Регистрация в муниципалитете → BSN
□ Открыть банк (ING, ABN AMRO, Bunq)
□ Health insurance (обязательна!)
□ Подать 30% ruling (4 месяца дедлайн!)
□ DigiD (государственный ID online)
□ OV-chipkaart (транспортная карта)
```

### Что компания обычно даёт

```
Standard:
□ Visa sponsorship
□ 30% ruling application
□ 1 month temporary housing

Good packages add:
□ Relocation bonus (€3-5K)
□ Moving costs
□ Tax advisor for 30% ruling
```

## Плюсы и минусы

### Плюсы

```
✓ Быстрая виза (2-4 недели)
✓ 30% ruling = существенная экономия
✓ Все говорят по-английски
✓ Work-life balance (реальный, не на бумаге)
✓ Cycle-friendly, качество жизни
✓ Путь к EU citizenship (5 лет)
✓ Strong tech scene
```

### Минусы

```
✗ Housing crisis — реально сложно найти жильё
✗ Погода (серо, дождливо, ветрено)
✗ Без 30% ruling налоги ~40-50%
✗ 30% ruling теперь только 5 лет (было 8)
✗ Childcare дорогой
```
