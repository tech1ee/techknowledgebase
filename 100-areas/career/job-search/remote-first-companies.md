# Remote-First компании: где искать полностью удалённую работу

Remote-first — это не "можно работать из дома иногда". Это компании, где удалёнка в ДНК: процессы, культура, компенсация построены вокруг распределённой команды. GitLab работает так с 2014 года, Automattic — с 2005. У них 2000+ сотрудников в 90+ странах и zero офисов.

## Красные флаги при поиске remote

### Что должно насторожить

```
❌ "Remote-friendly" вместо "Remote-first"
   → Офисные сотрудники имеют преимущество

❌ "Must overlap 8 hours with US timezone"
   → Это не remote, это ночная смена

❌ Нет информации о distributed team culture
   → Вероятно, remote как afterthought

❌ Очень низкая зарплата "для remote"
   → Эксплуатация geographic arbitrage

❌ "Contractor only" без опции full-time
   → Нет benefits, нестабильность
```

### Что искать в описании

```
✓ "Remote-first" или "Fully distributed"
✓ Async communication упомянут
✓ Home office stipend ($500-$1500)
✓ Co-working space reimbursement
✓ Clear timezone overlap requirements
✓ Remote работа в culture section
```

## Процесс найма в remote компаниях

### Типичные этапы

```
1. Application (async review, может занять 1-2 недели)
2. Recruiter Screen (Zoom, 30 min)
3. Technical Assessment (take-home или live coding)
4. Technical Interview (1-2 раунда, Zoom)
5. System Design (для senior)
6. Hiring Manager / Team Fit
7. Offer
```

### Особенности

- **Take-home чаще** — async culture распространяется на найм
- **Больше writing** — могут попросить написать doc/RFC
- **Culture fit важен** — remote требует self-discipline
- **Time zone указывают** — чаще US-overlap или EU-friendly

## Юридические аспекты

### Типы оформления

| Тип | Плюсы | Минусы |
|-----|-------|--------|
| **Full-time employee** | Benefits, stability | Нужна entity в твоей стране |
| **EOR (Employer of Record)** | Как employee, но через посредника | Немного дороже для компании |
| **Contractor** | Flexibility, higher gross | Сам платишь налоги, нет benefits |

### EOR-провайдеры (компания нанимает через них)

- **Deel** — популярный, 150+ стран
- **Remote.com** — конкурент Deel
- **Oyster** — фокус на developer experience
- **Papaya Global** — enterprise
