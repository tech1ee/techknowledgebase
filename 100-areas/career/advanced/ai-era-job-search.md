# AI-Era Job Search: как использовать AI и не быть отсеянным за него

77% job seekers уже используют AI в поиске работы. 75%+ компаний используют ATS с AI для скрининга. Но recruiters научились распознавать "AI-written" контент. Выигрывает тот, кто использует AI как tool, не как замену thinking.

---

## Ландшафт 2025

```
AI В JOB SEARCH:

Job Seekers используют AI для:
• Resume writing/optimization (63%)
• Cover letters (45%)
• Interview prep (38%)
• Job matching (63% — LinkedIn рекомендации)

Employers используют AI для:
• Resume screening (ATS) — 75%+ компаний
• Candidate matching
• Initial chatbot interviews
• Skills assessment

ПАРАДОКС:
AI помогает создавать контент →
Recruiters учатся распознавать AI →
AI-generated контент отсеивается →
Human touch становится differentiator
```

## AI Tools для Resume

### Оптимизация под ATS

```
TOOLS:

Jobscan (jobscan.co):
• Сравнивает resume vs JD
• Показывает match score
• Highlights missing keywords
• $49/month (есть free tier)

Teal (tealhq.com):
• Resume builder + job tracker
• AI suggestions for bullets
• ATS optimization
• Free tier достаточно мощный

Resume Worded (resumeworded.com):
• AI scoring резюме
• Line-by-line feedback
• LinkedIn review
• Free basic / $49/month pro

КАК ИСПОЛЬЗОВАТЬ:
1. Напиши resume сам
2. Прогони через tool
3. Добавь missing keywords
4. Перефразируй AI-suggestions своими словами
5. Проверь ещё раз

НЕ ДЕЛАЙ:
• Copy-paste AI-generated bullets целиком
• Keyword stuffing без смысла
• Одинаковый resume на все позиции
```

### ChatGPT для Resume

```
ХОРОШИЕ PROMPTS:

Rewrite bullet:
"Rewrite this resume bullet to be more impactful with metrics:
'Worked on Android app development'
Context: I built features for app with 2M users,
improved crash rate by 50%"

Match to JD:
"Here's a job description: [JD]
Here's my experience: [experience]
Suggest how to tailor my resume bullets to highlight
relevant skills for this role."

Optimize wording:
"Make this bullet more action-oriented:
'Responsible for implementing new features'"

ПЛОХИЕ PROMPTS:
• "Write me a resume"
• "Generate 10 bullets for Android developer"
• Anything без specific context

AI OUTPUT REQUIRES:
• Fact-checking (AI makes up metrics)
• Your voice (remove AI-isms)
• Personalization (generic → specific)
• Human review (sounds natural?)
```

## What AI Can and Cannot Do

```
AI CAN:
✓ Optimize keywords for ATS
✓ Improve bullet point wording
✓ Generate initial drafts
✓ Practice interview answers
✓ Research companies
✓ Summarize job descriptions

AI CANNOT:
✗ Know your actual experience
✗ Make you memorable
✗ Build genuine relationships
✗ Replace authentic voice
✗ Guarantee accuracy of claims
✗ Handle nuanced situations

RULE:
AI = efficiency tool, not replacement for thinking
```

## Avoiding AI Detection

```
RECRUITERS SPOT AI BY:

Red Flags:
• Overly perfect grammar
• Generic phrasing
• "AI words": leverage, utilize, synergize
• Lack of specific details
• Cookie-cutter structure
• No personality

HUMANIZE AI OUTPUT:

1. Add specifics:
   AI: "Improved app performance significantly"
   You: "Reduced startup time from 3.2s to 1.4s
         by implementing lazy initialization"

2. Add personality:
   AI: "Experienced in Android development"
   You: "Obsessed with smooth 60fps animations
         and pixel-perfect Compose layouts"

3. Remove AI-isms:
   Remove: leverage, utilize, spearheaded,
           passionate about, dynamic, innovative

4. Include failures/learnings:
   AI never includes this → human touch

5. Use specific examples:
   AI: "Strong problem-solving skills"
   You: "Debugged production crash affecting 15K users
         at 2am, shipped fix in 3 hours"
```

## AI-Enhanced Workflow

### Daily Job Search (30-60 min)

```
MORNING (15-20 min):
1. Check Otta/LinkedIn for new matches
2. Quick scan of 5-10 new postings
3. Save interesting ones to Notion/Teal

DEEP WORK (30-40 min):
1. Research ONE company deeply
2. Use ChatGPT to summarize their blog/news
3. Tailor resume для this company
4. Run through Jobscan for optimization
5. Write personalized outreach

TRACKING:
• All in one place (Notion/Teal/Spreadsheet)
• Status: Applied → Response → Interview → Offer
• Notes on каждой компании
```

### Interview Prep Workflow

```
DAY BEFORE:

Company Research:
1. Website, blog, recent news
2. ChatGPT: "Summarize [Company]'s recent product launches"
3. LinkedIn: Team members, hiring manager
4. Glassdoor: Interview questions, culture

Behavioral Prep:
1. Review your STAR stories
2. ChatGPT practice: 3-5 questions
3. Record yourself answering
4. Review and refine

Technical Prep:
1. Review their tech stack
2. Practice relevant coding
3. Prepare system design discussion
4. Questions for them

DAY OF:
• Light review, no cramming
• Test tech setup
• Prepare environment
• Deep breaths
```

## Common Mistakes

```
❌ Relying only on AI-generated content
   → Edit everything, add your voice

❌ Mass applying with AI speed
   → Quality >> quantity, even with AI help

❌ Trusting AI metrics/facts
   → AI invents numbers, always verify

❌ Using AI for everything
   → Networking still requires human touch

❌ Sounding like everyone else
   → AI makes everyone sound similar

❌ Ignoring AI tools entirely
   → Competitors are using them, you're at disadvantage

BALANCE:
AI для efficiency →
Human for creativity and connection →
Best results
```
